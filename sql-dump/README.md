# sql-dump (local only)

Put **ad-hoc queries, exports, and one-off scripts** here. This directory is gitignored except this file—nothing under `sql-dump/` is committed.

Use `sql-changes/` for migrations and `sql-changes/archive/ai-scripts/` for retained operational SQL that should stay in the repo.

All SQL that originated under **Project Docs** (including DDL) is in `imported-sql/project-docs-*.sql` here—local-only.

Non-DDL SQL from `oe_payment_manager` and `sharewell-csv-processor` is also under `imported-sql/` when it was not schema work.
