📧 Email Template Preview

Simple preview system - no server required!

🚀 How to Use:

  1. Click any template button below
  2. It will open the template in a new tab
  3. The template will show with sample data
  4. You can edit the URL parameters to test different values

🎯 Enrollment Invitation

Email sent when inviting someone to complete their benefits enrollment

🚀 Onboarding Invitation

Email sent when inviting someone to join and set up their account

👤 User Welcome

Email sent when a new user account is created

🎨 Customize Templates:

You can modify the URL parameters to test different values:

  • tenantName - Organization name
  • primaryColor - Main brand color (hex code)
  • secondaryColor - Secondary brand color (hex code)
  • accentColor - Accent color (hex code)
  • logoUrl - Logo image URL
  • memberFirstName - Recipient's first name
  • groupName - Group name (optional)

Example: Change memberFirstName=John to memberFirstName=Jane in the URL
